# Android_ESP8266_App1
The app is used to control ESP8266 remotely through a wifi connection. The app has two swtiches to turn on and off a remote source connected the the ESP8266 device. 
The corresponding ESP8266 code in the depository willl allow users to turn on and off a relay switch or LED connected to PIN 16

User of the app needs to connect to the ESP8266 which operates as a server with following IP address: http:https://192.168.4.1/ON and https://192.168.4.1/OFF

The app automatically sends the correct on/off command based on the swich that the app user is clicking
